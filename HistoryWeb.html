About This website

หน้าเว็บนี้เป็นหน้าที่เกี่ยวกับกิจกรรมต่างๆที่ผมทำในชีวิตประจำวันของผมครับ

My Activity

กิจกรรมยามว่างของผมก็จะเป็นการเล่นเกม ดูหนัง ฟังเพลง

เล่นเกม

กิจกรรมยามว่างส่วนใหญ่ของผมจะเป็นการเล่นเกมซึ่งเกมที่ผมเล่นเป็นส่วนใหญ่ก็คือ LoL,Valorant,Dota2

คลิกที่นี่

ดูหนัง

บางครั้งเวลาที่ผมว่าง ผมมักจะหาหนังดู และผมก็มีหนังที่ผมจะมาแนะนำ คลิกปุ่มด้านล่าง

คลิกที่นี่

ฟังเพลง

ผมเป็นคนที่ชอบฟังเพลงตอนว่างหรือตอนทำงาน ผมมีเพลงที่จะมาแนะนำให้ฟังซึ่งเป็นเพลงที่ฟังแล้วรู้สึกสงบ คลิกที่ปุ่มด้านล่าง

คลิกที่นี่

My Activity

นอกจากกิจกรรมด้านบนก็จะมีกิจกรรมอื่นๆ เช่น การตัดต่อคลิป การเขียนเว็บไซต์และโปรแกรม การวาดรูป

ตัดต่อคลิปวิดีโอ

การตัดต่อคลิปของผมจะใช้โปรแกรม After Effect และ Premiere Pro ซึ่งในการตัดต่อจะเน้นใช้ Effect เป็นส่วนใหญ่ คลิปตัวอย่างจะอยู่ที่ปุ่มด้านล่าง

คลิกที่นี่

เขียนเว็บไซต์และโปรแกรม

การเขียนเว็บไซต์ของผมจะใช้โปรแกรม Visual Studio Code และการเขียนโปรแกรมของผมจะใช้ Visual Studio ซึ่งในการเขียนโปรแกรมหลักๆตอนนี้จะเน้นไปที่การเขียนเว็บไซต์เป็นส่วนใหญ่

วาดรูป
